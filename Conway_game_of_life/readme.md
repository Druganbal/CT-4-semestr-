## Game of Life

Проект по моделированию колонии клеток (Conway's Game of Life).

## Описание

Программа моделирует развитие колонии клеток на двумерной решетке.
Каждое поколение вычисляется на основе правил игры Жизнь Конуэя.

## Цель проекта

Разработать программу на языке C, которая:
- моделирует поколения клеток
- отображает историю развития колонии
- эффективно работает с большими колониями
